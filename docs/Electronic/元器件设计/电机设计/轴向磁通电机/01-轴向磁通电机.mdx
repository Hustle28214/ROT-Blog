## 1. 轴向磁通电机的定义

轴向磁通电机，顾名思义，其磁通方向与电机转轴方向平行。它的结构像一个“煎饼”或“盘子”，定子和转子通常呈圆盘状，面对面排列，气隙磁场沿轴向穿过。

核心结构特点：

1. 盘式结构：电机外形扁平，直径远大于厚度，非常适合空间受限但对直径方向要求不高的场景。

2. 磁路走向：永磁体（在转子或定子上）的磁力线沿轴向穿过气隙和铁芯，构成回路。

3. 双转子单定子或单转子双定子：常见的有“双转子夹单定子”（中间是绕组，两边是转子盘）或“单转子夹双定子”结构，能有效利用磁场，提高功率密度。

## 2. 与径向磁通电机对比

我们通常所说的“正常电机”（如大部分工业电机、家用电器电机、电动汽车驱动电机）绝大多数是径向磁通电机。其磁通方向沿半径方向，定子在外围，转子在中心，像一个“圆筒”。

| 对比维度 | **轴向磁通电机** | **径向磁通电机（常见电机）** |
|:---|:---|:---|
| **磁路方向** | 与转轴**平行** | 与转轴**垂直** |
| **外观形态** | 扁平盘状（煎饼状）直径 > 厚度 | 圆柱状直径 < 厚度 |
| **结构布局** | 双转子单定子或单转子双定子面对面排列 | 外定子 + 内转子同心圆排列 |
| **功率/转矩密度** | **极高**（同等体积/重量下输出更大） | 相对较低（技术成熟但密度提升遇瓶颈） |
| **散热性能** | **优异**• 绕组端部短，热源近表面• 散热面积大，易冷却 | 相对较差• 绕组端部长，热量聚集• 需要复杂冷却系统（如水冷/油冷） |
| **转动惯量** | **低**薄盘转子，动态响应快 | 较高圆柱转子，响应相对较慢 |
| **材料与成本** | • 铁芯材料少（或无铁芯）• 永磁体用量多• 制造精度要求高• **成本较高** | • 材料用量相对均衡• 产业链成熟，自动化程度高• **成本低，经济性好** |
| **技术成熟度** | 新兴技术（近年快速发展） | 高度成熟（百年工业化应用） |
| **功率范围** | 中小功率应用为主（正向大功率发展） | 覆盖全面（瓦级 ~ 兆瓦级） |
| **轴向磁拉力** | 需要专门设计平衡（结构设计挑战） | 影响较小（技术处理成熟） |

## 3. 典型应用场景

### 轴向磁通电机适用领域
1. **电动汽车** - 轮毂电机、轮边电机、高性能车型主驱
2. **航空航天** - 飞行器推进系统、机载设备（对重量敏感）
3. **风力发电** - 直驱式发电机（可简化传动链）
4. **高端工业** - 机器人关节、高动态响应伺服系统
5. **特种设备** - 飞轮储能、医疗设备、精密仪器

### 径向磁通电机适用领域
1. **工业驱动** - 泵、风机、压缩机、传送带（90%+市场）
2. **家用电器** - 空调、洗衣机、冰箱、风扇
3. **电动汽车** - 主流车型主驱电机（当前主导技术）
4. **通用机械** - 从微型玩具电机到大型工业电机全覆盖
5. **基础设施** - 水泵、电站辅机、轨道交通牵引

## 4. 设计心得

笔者自己做过一个轴向磁通电机，包含仿真。制作这个东西最大的难点是磁铁，需要用到海尔贝克阵列（Halbach Array），这个东西不好弄到，也不好制作。

![](https://github.com/user-attachments/assets/31c821c5-d311-4419-8eff-698ce879c012)

要做的话，需要制作比较大的尺寸，不要尝试对标小型无刷电机那种，很容易烧。最好是从6020那个尺寸开始做。


![Image](https://github.com/user-attachments/assets/197a5af5-29a7-467e-b16c-f1a586046db6)

目前轴向磁通电机的生态尚不成熟，这个东西方案越多是越好的，性能还没有到顶，是非常有发展前景的技术。暂时形不成垄断。

虽然理论上散热好，但是你实际上用的时候，很难找到一个大片的面积用来散热，发热还是会比较密集的。

个人觉得用Ansys R1直接麦克斯韦电磁仿真比专用的电机CAD好用。间隙要留大一点，这个是失败作。
