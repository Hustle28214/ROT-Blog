高解析多媒体影音接口(High Definition Multimedia Interface, HDMI)是2002年由索尼、松下、日立等7家公司共同制定的标准，从2002年的HDMI1.0迭代到了2025年的HDMI2.2，是一个应用非常广泛的音视频流接口标准。

最初支持1080p/60Hz高清视频和8声道音频，到了2025年的HDMI2.2这一代，支持12K@120Hz、16K@60Hz等超高清标准。
