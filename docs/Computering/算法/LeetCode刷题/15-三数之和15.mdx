## 题干

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

示例 1：
```
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
解释：
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
```
注意，输出的顺序和三元组的顺序并不重要。

示例 2：
```
输入：nums = [0,1,1]
输出：[]
解释：唯一可能的三元组和不为 0 。
```
示例 3：
```
输入：nums = [0,0,0]
输出：[[0,0,0]]
解释：唯一可能的三元组和为 0 。
```

## 思路

`nums[i]+nums[j]+nums[k]=0`，很明显三个数暴力搜索是不好做的，所以用数学上的方法，移项，问题转化为`nums[i]+nums[j]+nums[k]=0`。

设置一个固定的`nums[i]`，这个`nums[i]`需要遍历，此过程中用双指针进行搜索。

由于题目有不重复要求，需要保证`nums[i]`，`nums[j]`，`nums[k]`不重复。使用排序去重。

双指针，设置左边一个指针，右边一个指针，如果他们两个

## 题解

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        int pl = 0;
        int pr = 0;
        vector<vector<int>> ans;
        sort(nums.begin(),nums.end());

        
        for(int i = 0; i<nums.size();++i){
            if (i > 0 && nums[i] == nums[i-1]) continue;
            pl = i + 1;
            pr = nums.size() - 1;
            
            while(pl<pr){
                if(nums[pl]+nums[pr]+nums[i]==0){
                    ans.push_back({nums[i], nums[pl], nums[pr]});

                    while (pl < pr && nums[pl] == nums[pl+1]) pl++;
                    while (pl < pr && nums[pr] == nums[pr-1]) pr--;
                    pl++;
                    pr--;
                }
                else if(nums[pl]+nums[pr]+nums[i]<0){
                    pl++;
                }else{
                    pr--;
                }
            }
            
            
        
        }
        return ans;
    }
};
```
