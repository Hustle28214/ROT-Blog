## 协程Coroutines

协程（Coroutines）是一项用于编写异步、惰性生成、流水线处理等逻辑的语言特性。

:::tip
协程是一种特殊的函数，它可以在执行过程中暂停和恢复。暂停时，协程会保存其当前的执行状态，包括局部变量、寄存器状态等。恢复时，协程会从上次暂停的位置继续执行。
:::

协程是一种可挂起、可恢复的函数。

普通函数一旦调用，就会一直运行到 `return` 或函数结束；协程则可以在执行中途通过 `co_await`、`co_yield` 或 `co_return` 挂起，把控制权交还给调用者，稍后再从挂起点继续执行。

在 C++20 中，协程是无栈协程（stackless coroutine）——它的状态（局部变量、挂起点、当前进度）被分配在堆上的一个协程帧中，而不是占用独立的系统栈。这使其非常轻量，可以创建成千上万个协程而不影响性能。
