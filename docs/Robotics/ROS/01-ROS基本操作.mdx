## 概念

作为操作系统，ROS(Robot Operating System，机器人操作系统)是一种底层抽象。它更像是一种框架，简化了机器人任务的编写过程。

四大门类：

1. Plumbing: 通讯机制（实现ROS不同节点之间的交互）
2. Tools: 工具软件包（ROS中的开发和调试工具）
3. Capabilities: 机器人高层技能（ROS某些功能的集合，比如导航）
4. Ecosystem: 机器人生态（ROS中各种功能模块的集合）
