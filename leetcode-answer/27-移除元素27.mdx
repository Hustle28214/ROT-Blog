## 题干

给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素。元素的顺序可能发生改变。然后返回 nums 中与 val 不同的元素的数量。

假设 nums 中不等于 val 的元素数量为 k，要通过此题，您需要执行以下操作：

- 更改 nums 数组，使 nums 的前 k 个元素包含不等于 val 的元素。nums 的其余元素和 nums 的大小并不重要。
- 返回 k。

用户评测：

```cpp

for (int i = 0; i < actualLength; i++) {
    assert nums[i] == expectedNums[i];
}
```

## 思路

:::tip
多余的元素，能不能直接删除？或就像83删除链表重复元素那样跳过？

数组的元素在内存地址中是连续的，不能单独删除数组中的某个元素，只能覆盖。
:::

暴力解法是最容易想的，遍历一遍数组，只要找到相等的元素，就把目标值之后的元素挪到前面一位来。

双指针法是更进一步，

## 题解

暴力：

```cpp
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int numSize = nums.size();
        for(int i=0;i<numSize;++i){
            if(nums[i]==val){
                for(int j=i+1;j<numSize;++j){
                    nums[j-1] = nums[j];
                }
                i--;//复位i
                numSize--;//现在相当于删掉了一个元素
            }
            
        }
        return numSize;
    }
};
```

快慢指针（要加深理解）：

```cpp
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int numSize = nums.size();
        int j = 0;// 慢指针，指向下一个要存放非 val 元素的位置
        for(int i=0;i<numSize;++i){
            if(nums[i]!=val){
                nums[j++] = nums[i];// 将非 val 元素复制到慢指针位置，并移动慢指针
            }
            // 如果是val元素，跳过，不复制也就被非val元素覆盖了
        }
        return j;// j 就是新长度，即非 val 元素的个数
    }
};
```
