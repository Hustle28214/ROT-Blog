## 题干

给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 `O(n)` 的算法解决此问题。

 

示例 1：
```
输入：nums = [100,4,200,1,3,2]
输出：4
```
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。

示例 2：
```
输入：nums = [0,3,7,2,5,8,4,6,0,1]
输出：9
```
示例 3：
```
输入：nums = [1,0,1,2]
输出：3
```
 

提示：
```
0 <= nums.length <= 10^5
-10^9 <= nums[i] <= 10^9
```

## 思路

用哈希表（unordered_set）来记录每个元素是否出现过。

:::danger
在最长连续序列算法中，检查 i-1 是否存在于集合中，是为了判断当前元素 i 是否可以作为某个连续序列的起点。

注意这里的i不是指数组下标！而是比如说数组元素100，i-1指的就是99！一定要分清楚这个概念，事实上明确了这个概念后，代码怎么写就很好想了。
:::
