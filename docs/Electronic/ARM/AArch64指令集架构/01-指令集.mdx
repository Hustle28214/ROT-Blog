## 指令集架构(ISA)

指令集架构(ISA)是计算机抽象模型的一部分。它定义了软件如何控制处理器。 

Armv8-A 支持三个指令集:A32、T32 和 A64。 有两种主要执行状态：

<Mermaid
value={`graph TB
    A[ARMv8-A 架构] --> B[AArch64 执行状态]
    A --> C[AArch32 执行状态]
    
    B --> B1[使用 A64 指令集]
    B1 --> B2[固定长度 32位指令]
    
    C --> C1[使用 A32/T32 指令集]
    C1 --> C2[向后兼容 ARMv7]`}
/>


指令执行流程:

<Mermaid
value={`graph LR
    A[内存中的指令序列] --> B[取指 Fetch]
    B --> C[解码 Decode]
    C --> D[执行 Execute]
    D --> E[写回 Write-back]
    
    style A fill:#e3f2fd,stroke:#1565c0`}
/>

这是程序员看到的"抽象机器"行为.
