## 2.1 C++程序。

C++程序包括基本的三部分：

1. 预处理部分
2. 全局声明部分
3. 函数

### 2.1.1 预处理部分

在C++程序中，开始位置处以"#"开头的部分。

预处理命令一般有三类：

1. 宏定义命令
2. 文件包含命令
3. 条件编译命令

#### 宏定义命令

C++一般通过#define指令进行宏定义，其包括不带参数的宏定义和带参数的宏定义。

```cpp
//不带参数的宏定义
#define PI 3.1415926
//带参数的宏定义
#define SQUARE(x) x*x
```

宏定义的优先级比变量定义高，如果宏定义和变量定义重名，则优先使用宏定义。
